Yabe & Bonnie 一頁式網站

檔案內容：
- index.html：網站首頁
- assets/hero-dogs.jpg：首頁照片
- assets/yabe-bonnie-logo.jpg：品牌圖片
- vercel.json：Vercel 設定

Vercel 上傳方式：
1. 解壓縮 ZIP。
2. 進入 Vercel Dashboard。
3. 選擇 Add New → Project。
4. 選擇「Import Third-Party Git Repository」或使用 GitHub 建立專案。
5. 將整個資料夾上傳，確認首頁檔案名稱是 index.html。
6. 點 Deploy。
